# Tiny Language Model

> *"I didn't build this because it was the easiest way. I built it because I wanted to understand how it works."*

This project is my journey of learning how language models work from the ground up.

I'm not trying to build the next GPT or train the biggest model. My goal is to understand every piece myself instead of treating it like a black box.

## What I'm learning

* Tokenization
* Vocabulary
* Word embeddings
* Next-word prediction
* Attention
* Transformers
* Training and inference

## Why this project?

There are already amazing pretrained models that can do incredible things.

I chose a different path.

Instead of adapting an existing model, I wanted to learn how one is built. This repository is where I experiment, make mistakes, rewrite code, and slowly understand the math behind modern AI.
